# Experiment 4: DOM Manipulation and jQuery

This project demonstrates DOM manipulation and jQuery techniques using a student information card interface.

## Features
- **Add Student**: Input student name and course, display in a styled card with fade-in effect
- **Hide/Show Card**: Toggle card visibility with dedicated buttons
- **Toggle Card**: Switch card visibility state
- **Animate Card**: Smoothly expand card dimensions and adjust opacity

## Technologies
- HTML5, CSS3, jQuery 3.6.0
- Responsive button interactions with hover effects
- Animated transitions and transformations
